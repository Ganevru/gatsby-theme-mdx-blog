
# Notes

## Theme Shape

- colors
- ...system theme
- layout/blocks
  - root
  - header
  - main
  - sidebar
  - container
  - footer
- mdx/styles/components
  - h1
  - h2
  - p
  - img
  - a
  - etc...

## Components

```jsx
<ThemeProvider>
  <Layout>
    <Header />
    <Main>
      <Content />
    </Main>
    <Footer />
  </Layout>
</ThemeProvider>
```

```jsx
<ThemeProvider>
  <Layout>
    <Header />
    <Main>
      <Sidebar />
      <Container>
        <Content />
      </Container>
    </Main>
    <Footer />
  </Layout>
</ThemeProvider>
```

Layout handles:

- Base typography
- colors
- layout for flexbox/etc

Alternative names:

- Root

```jsx
<ThemeProvider>
  <Root>
    <Header />
    <Main>
      <Container>
        <Content />
      </Container>
    </Main>
    <Footer />
  </Root>
</ThemeProvider>
```

```jsx
<Layout.Root>
  <Layout.Header />
  <Layout.Main />
  <Layout.Footer />
</Layout.Root>
```
